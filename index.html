About

I am a gamer and programmer from Australia in my mid 20s. I go by the gamertag "HellaMadMax" and mainly play FPS games such as Arma 3, Hell Let Loose, Call of Duty, Counter-Strike, etc.
I do programming as a full-time job with Visual FoxPro but have used other programming (and markup) languages including:

  • HTML and CSS
  • JavaScript (including NodeJS)
  • PHP
  • SQL (mainly MySQL/MariaDB)
  • LUA (Garry's Mod)
  • SourcePawn (Source Mod Plugins)

I used to host Day of Defeat: Source and Plutinum Call of Duty Zombies servers but lost interest in them.
You can find my socials at the bottom right of the page.

Projects

Below are a few personal projects I've made public to hopefully be used/referenced by others looking for the same thing.

  • gamedig-discord-status - nodejs script for hosting a discord bot that shows player count and can send messages when server is unresponsive.
  • sourcemod-plugins - few scripts I made for sourcemod (mostly for Day of Defeat: Source).
  • DepotDownloader - fork of DepotDownloader that allows -pubfile and -ugc to have multiple id's. useful for updating lots of big workshop mods that timeout with normal steamcmd.

Other old or redundant projects can be found on my github.
